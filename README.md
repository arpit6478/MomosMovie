# Momo's Movie - Movie Streaming Platform

A complete Telegram-based movie automation system with frontend website, backend API, and Telegram bot admin panel.

## Features

### Frontend Website
- Mobile-first responsive design
- Dark theme (Netflix style)
- Movie listing with filters
- Movie detail pages
- 5-second delay before redirect
- Fake "Get Link" button
- Ad placements (banner, native, sticky)

### Backend API
- Node.js + Express
- Supabase database integration
- RESTful API endpoints
- CORS enabled

### Telegram Bot
- Admin-only access
- Button-based UI
- Add movies with 7-step flow
- Auto-post to Telegram channel
- View statistics
- Delete movies
